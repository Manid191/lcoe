# Solar LCOE & PPA Calculator (Portable Edition)

This is a lightweight, single-page application for Solar Power Plant Financial Analysis.

## Features
- **LCOE Calculation:** Levelized Cost of Energy analysis.
- **PPA & Cash Flow:** Detailed cash flow projections over 20-25 years.
- **Portable:** Runs entirely in the browser. No server required.
- **Configurable:** Edit `config.js` to change default assumptions (Financial, Technical).

## How to Use
1. **Open `index.html`** in any modern web browser.
2. **Edit Defaults:** Open `config.js` in a text editor to change startup values.
3. **Save/Load:** Use the "Save File" button in the app to save your scenarios as `.json` files.

## Deployment (GitHub Pages)
1. Upload `index.html` and `config.js` to your GitHub repository.
2. Enable GitHub Pages in Settings -> Pages.
3. Access your calculator via the provided GitHub Pages URL.
